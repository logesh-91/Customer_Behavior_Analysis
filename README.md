# Customer_Behavior_Analysis
data analytics project showcasing customer shopping behavior using python, sql and power bi.
# 📊 Data Analytics Project README

## 🔍 Overview

This project demonstrates an end-to-end **data analytics workflow**, starting from data loading to generating business insights. It covers data preprocessing, exploratory data analysis (EDA), SQL-based querying, dashboard creation, and presentation of insights.

The goal is to transform raw data into meaningful insights using industry-standard tools and practices.

---

## 📁 Dataset

* The dataset contains structured data used for analysis.
* Includes multiple attributes such as categories, numerical values, and identifiers.
* Data quality issues like missing values, duplicates, and inconsistencies were handled during preprocessing.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, and EDA
* **MySQL Server** – Data storage and querying
* **SQL** – Data extraction and transformation
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) creation

---

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Verified structure, data types, and initial records

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column names and formats
* Validated data consistency

### 3. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships
* Identified trends, patterns, and outliers
* Used visualizations for better understanding

### 4. SQL Analysis (MySQL)

* Loaded cleaned data into MySQL
* Performed queries 

### 5. Dashboard Development (Power BI)

* Built interactive dashboards
* Included KPIs, filters, and visual insights
* Focused on clarity and business usability

### 6. Presentation (Gamma)

* Created a clean and concise presentation
* Highlighted:

  * Problem statement
  * Approach
  * Key insights
  * Business recommendations
